实验总结：

通过本次实验，熟悉了awk、tr、sed命令的使用方法。尤其对awk命令有了更深的理解。

在实践过程中发现.awk格式文件中，并没有对’+‘运算符重载，也就是并不支持两个字符串以“a”+”b”的形式进行连接。如果使用c = “a”+”b”这种形式，会将a、b先转为数字类型，然后进行加法，c的类型就会是一个数字；如果要实现字符串连接功能，形式如下：

​ a = “123”

​ b = “456”

​ c = (ab)

​ 则 c 为”123456”
